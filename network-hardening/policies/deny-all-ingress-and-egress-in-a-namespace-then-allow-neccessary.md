## Deny All Ingress & Egress Policy
```yaml
## vi deny-all-traffic-default-ns.yaml
---
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: deny-all-ingress-egress-in-default-ns-only
  namespace: default
spec:
  podSelector: {}
  policyTypes:
  - Ingress
  - Egress
```
```sh
kubectl apply -f deny-all-traffic-default-ns.yaml
```
